# StardewModsSolution
Main repo with the solution for my Stardew Valley mods

## Init repositories (projects)

In the bash (linux bash on native linux or WSL or git bash mingw on Windows) run command:

```bash
bash init.sh
```

## Update

In the bash (linux bash on native linux or WSL or git bash mingw on Windows) run command:

```bash
bash update.sh
```
